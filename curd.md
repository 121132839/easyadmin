| 字段类型 | 说明 |类型 | 默认值| 备注 |
| --- | --- |--- |--- |--- |
| upload | 上传文件 | string | null |  |
| select | 下拉选择 | string | null |  |
| checkbox | 复选框多选 | bool | false | |
| radio | 单选框单选 | int | 0 |  |
| switch | 开关类型单选 | int | 0 |  |
| time | 时间类型 | int | 0 |  |
| textarea | 文本框 | int | 0 |  |
| editor | 富文本编辑器 | int | 0 |  |
